# HCI Project (Human-Computer Interaction)

> Project developed within the scope of the Human-Computer Interaction course unit.
> The application features a monitoring platform for the Recovery and Resilience Facility, allowing users to consult information regarding Member States, common indicators, investment pillars, financial execution, targets, and milestones.

> Final grade: 19.55/20 ⭐


![main page](relatório/main-page.png)

![footer](relatório/footer-img.png)

## Main Features

- Overview of European implementation.
- Country consultation and respective details.
- Comparison between two countries.
- Analysis of pillars and common indicators.
- Overview of disbursements by country.
- Visualization of targets, milestones, and history.
- FAQ section.

## Technologies Used

- Vue
- Vite
- TypeScript
- Vue Router
- Chart.js
- JSON Server

### How to Run the Project

Inside the projeto-ipm directory, open two separate terminals and run the following commands:

- `Terminal 1`: npm run server
- `Terminal 2`: npm run dev
